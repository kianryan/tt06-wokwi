## How it works

Two XOR gates input a AND gate.  The the output from the AND gate is the final OUTPUT.

## How to test

XOR 1 is on INPUT 0/1, XOR 2 is on INPUT 2/3.
OUTPUT 0 is OUTPUT from AND.
